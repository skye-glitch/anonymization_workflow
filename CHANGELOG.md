## [0.1] 2025-08-28

### Added

- This repository contains marimo notebooks that utilize the Presidio Framework and a spacy model to anonymize help desk tickets.
- README.md describes how to edit and run the notebooks.
- The `annotation/` directory contains a notebook that converts annotated tickets in json format to spacy Doc files.
- The `analyzer/` directory contains a notebook and sample databases to create a Presidio analyzer.
- The `evaluator/` directory contains a notebook to create a Presidio evaluator.
- The `training/` directory contains a notebook to train a custom spacy model using spacy Doc files.
- The `anonymizer/` directory contains a notebook to create a Presidio anonymizer.
